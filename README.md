# RAG Chatbot

## Overview
This is a Retrieval-Augmented Generation (RAG) chatbot that uses a vector database for context retrieval and a language model for generating responses.

## Features
- Context-aware answers
- Custom dataset ingestion
- Supports local or API-based LLMs

## Installation
```bash
git clone https://github.com/Pranay9-coder/rag-chatbot.git
cd rag-chatbot
pip install -r requirements.txt
